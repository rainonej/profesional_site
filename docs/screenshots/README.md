## How to add screenshots

1. Take a screenshot of each item listed as ⬜ Needed above
2. Crop to the relevant area (no need to capture the full browser chrome)
3. Save as PNG using the filename in the table
4. Commit and push — images appear inline in CONTRIBUTING.md automatically
